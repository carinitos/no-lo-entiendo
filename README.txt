"no lo entiendo" //a project by ya boi

https://substack.com/home/post/p-162057759

application that generates cut-up song lyrics similar to the style of William Burroughs.
also generates chords, tempo and capo usage. 

How to Use:
upload material to be "cut-up" into input directory
filename of input file doesnt matter
program reads the file and does the work
lyrics are written to a txt file in the output directory using the title you provided
output is 100 random sentences, key, chord pattern w/ chord names, and bpm
up to user to create a cohesive song using these pieces 
sample output attached in output folder
be creative, have fun :) 

NOTE: 
no generative AI was used during the creation of this application
this program is not intended to be used with any copywritten material. 
have fun, don't be a dweeb
